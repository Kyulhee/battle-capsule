## Battle Capsule v0.1.0 — Closed Alpha

Quarter-view battle royale prototype. 1 player vs 11 bots, one forest map.

**Controls**
| Key | Action |
|---|---|
| WASD | Move |
| Mouse | Aim |
| Left Click | Shoot |
| E | Pick up item |
| Q | Heal |
| F12 | Debug screenshot |

**What's in**
- Zone shrink (blue zone)
- Loot & weapon swap (AR / Shotgun / Railgun)
- Bot AI with stealth/perception system
- Supply drop at zone stage 2
- Match history (Records screen)

**Known limitations**
- Placeholder audio (procedural beeps, no real SFX yet)
- No animations
- Single map, forest theme
